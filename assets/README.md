# Assets Directory

This directory contains all application assets:

- `images/` - App images and graphics
- `icons/` - Custom icons
- `filters/` - Image filter presets
- `models/` - AI/ML model files

## Images
Place your app images here.

## Icons
Place custom icon assets here.

## Filters
Store filter configuration and lookup tables here.

## Models
Store TensorFlow Lite models and other ML assets here.
