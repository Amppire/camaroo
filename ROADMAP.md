# Roadmap

This document outlines the future development plans for Camaroo.

## Version 1.0.0 ✅ (Current Release)

### Features
- ✅ Professional camera controls
- ✅ Manual settings (ISO, shutter speed, exposure, focal length)
- ✅ Portrait mode
- ✅ Grid overlay and histogram
- ✅ Photo editor with filters and adjustments
- ✅ Gallery with albums
- ✅ AI-powered features (face detection, scene recognition)
- ✅ Cross-platform support (iOS & Android)

## Release Schedule

- **Minor Versions** (1.x.0): Every 3 months
- **Patch Versions** (1.0.x): As needed for bug fixes
- **Major Versions** (x.0.0): Annually

## Development Priorities

Our development is guided by these principles:

1. **Stability First**: New features must not compromise app stability
2. **User Privacy**: Privacy-focused features over data collection
3. **Performance**: Optimize before adding more features
4. **Open Source**: Maintain transparent development
5. **Community Driven**: Listen to user feedback
